Fase 3: Usuaris, Grups i Permisos

Usuari Anonymous

  • Directori Home: c:\ftp\pub
  • Contrasenya: No requerida.
  • Permisos: Només **Lectura** (Descàrrega).
  • Grup: Cap.

Usuari Professor

  • Directori Home: c:\ftp
  • Contrasenya: Requerida.
  • Permisos: **Lectura, Escriptura, Esborrar, Modificar** (Control Total).
  • Grup: Cap.

Grup Alumnes (Alumne1, Alumne2)

  • Directori Home: c:\ftp\alumnes
  • Permisos: **Lectura** i **Escriptura**.
  • Límit Pujada: 2 KB/s (Heretat del Grup).
  • Autenticació: Contrasenya individual.

Fase 4: Mode Passiu i Connexió Segura (FTPS)

Configuració del Mode Passiu

  • Utilitat: Resoldre problemes de NAT/Firewall.
  • IP Ús: Definir IP específica del Servidor/Client.
  • Rang de Ports: Personalitzat i obert al Firewall (Ex: 5000 a 5100).

Seguretat Amb SSL/TLS (FTPS)

  • Activació: Marcar "Enable FTP over SSL/TLS support".
  • Certificat: Generar un nou certificat auto-signat (Key i CRT).
  • Port Segur: Utilitzar port alternatiu per a connexions segures (Ex: 990).
  • Usuari Segur: Marcar "Require a secure connection" per forçar FTPS.
